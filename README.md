# 🛡️ Network Security Project

Ce dépôt GitHub contient deux parties principales liées à la cybersécurité défensive et aux opérations SOC.

---

# 🔹 1. Infrastructure Sécurisée + Mini SOC

La première partie du projet consiste en l’implémentation d’une infrastructure réseau sécurisée accompagnée d’un mini SOC basique.

## Contenu principal

* Segmentation réseau
* Configuration de pare-feu
* Surveillance basique des événements
* Centralisation des logs
* Détection simple d’activités suspectes
* Architecture sécurisée virtualisée

## Objectif

Mettre en place un environnement réseau sécurisé permettant d’introduire les bases du monitoring et de la détection d’incidents.

---

# 🔹 2. SOC Open Source Opérationnel

La deuxième partie du dépôt contient un SOC Open Source complet et opérationnel.

## Technologies utilisées

* Wazuh
* TheHive
* Zeek
* Suricata
* Docker
* Linux
* FortiWeb

## Fonctionnalités

* SIEM Open Source
* Détection d’intrusions
* Analyse réseau
* Gestion des incidents
* Corrélation des événements
* Automatisation des alertes
* Détection d’attaques avancées
* Intégration Blue Team

## Objectif

Reproduire une architecture SOC réaliste permettant la supervision, la détection et la réponse aux incidents dans un environnement virtualisé.
